# News Text Classification (Arabic)

Machine learning project for classifying Arabic news articles into predefined categories. The work is organized as Jupyter notebooks and includes data preprocessing, model training, and evaluation.

## Repository structure

- `arabic_news_classification.ipynb` — main notebook (preprocessing, training, evaluation)
- `sample.ipynb` — additional experiments
- `requirements.txt` — Python dependencies
- `*.png` — generated figures and visualizations

## Requirements

- Python 3.9+ (recommended)
- Jupyter Notebook or JupyterLab

Install dependencies:

```bash
pip install -r requirements.txt
```

## How to run

1. Create and activate a virtual environment (recommended).
2. Install the requirements.
3. Start Jupyter.

```bash
jupyter lab
```

4. Open `arabic_news_classification.ipynb` and run the cells in order.

## Notes

- The notebooks contain the full workflow, including text cleaning and normalization steps specific to Arabic.
- Figures (`.png`) are generated by the notebooks during analysis.
