# News Text Classification (Arabic)

This project classifies Arabic news articles into predefined categories using classical machine learning models. The work is organized as Jupyter notebooks covering data loading, preprocessing, feature extraction, model training, and evaluation.

## Repository Contents

- `arabic_news_classification.ipynb` — main notebook for preprocessing, training, evaluation, and visualization
- `sample.ipynb` — additional experiments
- `requirements.txt` — Python dependencies
- `*.png` — figures generated by the notebooks (confusion matrix, EDA plots, model comparison)

## Dataset

The notebook expects a CSV file named `arabic_news.csv` in the repository root (the same directory as the notebooks) with at least the following columns:

- `text` — the Arabic news article text
- `label` — the category label

The notebook describes three target categories:

- رياضة (Sports)
- سياسة (Politics)
- اقتصاد (Economy)

The dataset file is not included in this repository. Provide your own dataset in the required format before running the notebook.

## Requirements

- Python 3.9+ (recommended)
- Jupyter Notebook or JupyterLab

Install dependencies:

```bash
pip install -r requirements.txt
```

## How to Run

1. (Recommended) Create and activate a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Start Jupyter:

```bash
jupyter lab
```

4. Open `arabic_news_classification.ipynb` and run the cells in order.

## Methods Used

The notebook compares multiple traditional machine learning approaches:

- Feature extraction: Bag of Words (CountVectorizer) and TF-IDF
- Models: Logistic Regression, Linear SVM (LinearSVC), Multinomial Naive Bayes, Random Forest

## Outputs

Running the notebook generates the following artifacts in the repository root:

- `confusion_matrix.png`
- `eda_plots.png`
- `model_comparison.png`

## Notes

- The preprocessing pipeline includes Arabic-specific text normalization and cleaning steps.
- Model results depend on the dataset you provide and may vary.
